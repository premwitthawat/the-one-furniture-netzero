# วิธีแก้ไขและอัปโหลดเว็บไซต์

เว็บไซต์นี้เป็นไฟล์เดียว (`index.html`) ไม่ต้องใช้ฐานข้อมูลหรือระบบหลังบ้าน เปิดดูได้ทันทีด้วยการดับเบิลคลิกไฟล์

## แก้ไขเนื้อหา (ตัวเลข ข้อความ มาตรการ)

1. เปิดไฟล์ `index.html` ด้วยโปรแกรมแก้ไขข้อความ (Notepad, VS Code, หรือ TextEdit ก็ได้)
2. กด Ctrl+F (หรือ Cmd+F) ค้นหาคำว่า **`แก้ไขข้อมูลเว็บไซต์ตรงนี้`**
3. จะเจอบล็อกใหญ่ชื่อ `const SITE = { ... }` — ทุกข้อความ/ตัวเลขในเว็บไซต์อยู่ในนี้ทั้งหมด
4. แก้ไขเฉพาะข้อความในเครื่องหมายคำพูด `"..."` ได้เลย เช่น
   ```js
   { name: "LED + เซ็นเซอร์...", goal: "ลดไฟแสงสว่าง 40-60%", ... status: "planned" }
   ```
5. เปลี่ยนสถานะมาตรการเมื่อเริ่มลงมือทำจริง: `status` มี 3 ค่า
   - `"planned"` = แผน (สีทอง)
   - `"progress"` = กำลังดำเนินการ (สีเขียวอ่อน)
   - `"done"` = เสร็จสิ้น (สีเขียวเข้ม)
6. จะเพิ่มมาตรการใหม่ 1 ข้อ ให้ copy ทั้งก้อน `{ name: ..., goal: ..., ... }` ของรายการที่ใกล้เคียง วางต่อท้ายในลิสต์เดียวกัน (อย่าลืมคอมม่าคั่นระหว่างรายการ) แล้วแก้ข้อความข้างในทีหลัง
7. บันทึกไฟล์ แล้วเปิดดูใหม่ในเบราว์เซอร์เพื่อตรวจสอบ

**ข้อควรระวัง:** อย่าลบเครื่องหมาย `{ }`, `[ ]`, `,` หรือ `"` — ถ้าโครงสร้างเพี้ยน เว็บอาจไม่ขึ้นข้อมูล ให้ลองใช้ Ctrl+Z ย้อนกลับแล้วแก้ใหม่

## เพิ่มลิงก์ดาวน์โหลดรายงานจริง

ในส่วน `documents:` เปลี่ยนค่า `href: "#"` เป็นลิงก์จริงหลังจากอัปโหลดไฟล์รายงาน (.docx/.pdf) ขึ้นที่เก็บไฟล์ เช่น Google Drive (แชร์แบบ "ทุกคนที่มีลิงก์"), เว็บไซต์บริษัท หรือ GitHub

## นำขึ้น GitHub (เก็บโค้ดออนไลน์)

1. ไปที่ **https://github.com/signup** (ถ้ายังไม่มีบัญชี) หรือ **https://github.com/login** (ถ้ามีบัญชีแล้ว)
2. หลังล็อกอินแล้ว ไปที่ **https://github.com/new**
3. กรอกข้อมูล:
   - **Repository name**: เช่น `the-one-furniture-netzero`
   - เลือก **Public**
   - **ไม่ต้อง**ติ๊ก "Add a README file" (เดี๋ยวเราอัปโหลดของเราเอง)
   - กดปุ่มสีเขียว **Create repository**
4. หน้าถัดไปจะมีข้อความสีฟ้าเขียนว่า **"uploading an existing file"** อยู่กลางหน้า — คลิกตรงนั้น
5. ลากไฟล์ `index.html` และ `README.md` จากเครื่องมาวางในกล่อง "Drag files here"
6. เลื่อนลงล่างสุด กดปุ่มสีเขียว **Commit changes**
7. เสร็จแล้ว โค้ดจะอยู่ที่ `https://github.com/ชื่อบัญชีของคุณ/the-one-furniture-netzero`

## นำขึ้น Netlify (ให้ได้ลิงก์เว็บไซต์ใช้งานจริง)

**วิธีแนะนำ — เชื่อมกับ GitHub** (ข้อดี: แก้ไฟล์ทีหลังแล้วเว็บอัปเดตให้อัตโนมัติ ไม่ต้องอัปโหลดใหม่เอง):

1. ไปที่ **https://app.netlify.com/signup** เลือก **"Sign up with GitHub"** (ใช้บัญชี GitHub เดียวกับขั้นตอนก่อนหน้า ไม่ต้องตั้งรหัสผ่านใหม่)
2. กด **"Add new site"** → **"Import an existing project"**
3. เลือก **"Deploy with GitHub"** แล้วกด **Authorize** เพื่ออนุญาตให้ Netlify เข้าถึง repository
4. เลือก repository ที่เพิ่งสร้าง (`the-one-furniture-netzero`)
5. ช่อง Build command **ปล่อยว่างไว้** (เว็บนี้เป็น HTML ล้วน ไม่ต้อง build)
6. Publish directory ปล่อยเป็นค่า default
7. กด **Deploy site** รอประมาณ 30 วินาที จะได้ลิงก์ เช่น `https://random-name-12345.netlify.app`
8. ถ้าอยากได้ชื่อลิงก์ที่จำง่ายกว่า: **Site settings → Change site name**

หลังจากนี้ ทุกครั้งที่แก้ไฟล์ใน GitHub (แก้ผ่านเว็บ GitHub ตรงๆ ก็ได้ ไม่ต้องมีโปรแกรมอะไรเพิ่ม) เว็บไซต์บน Netlify จะอัปเดตให้เองภายในไม่กี่นาที — กดไอคอนดินสอ (Edit this file) มุมขวาบนของไฟล์ `index.html` ใน GitHub แก้ตรงนั้นได้เลย

**วิธีลัด — ไม่ต้องมีบัญชีเลย** (ได้ลิงก์ทันที แต่แก้ไขทีหลังต้องอัปโหลดไฟล์ใหม่ทุกครั้ง):

1. ไปที่ **https://app.netlify.com/drop**
2. ลากไฟล์ `index.html` มาวาง
3. ได้ลิงก์ใช้งานได้ทันที

## หมายเหตุสำคัญ

เนื้อหาทั้งหมดในเว็บไซต์นี้แปลงมาจากรายงานภาคเรียน (Sustainable & Green Profit from Net Zero and Carbon Management) มาตรการทุกข้อยังอยู่ในสถานะ **"แผน"** เพราะยังไม่ได้เริ่มดำเนินการจริง แนะนำให้ทยอยอัปเดตสถานะและตัวเลขจริงเมื่อบริษัทเริ่มลงมือทำ เพื่อความน่าเชื่อถือของข้อมูลที่เผยแพร่
